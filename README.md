# Социальная сеть - API сервис

## Тема
Социальная сеть для публикации постов, лайков и подписок.

## Основные сущности
- **User** — пользователь
- **Post** — пост
- **Comment** — комментарий
- **Like** — лайк
- **Follow** — подписка

## Операции сервиса

### CRUD (для User, Post, Comment)
- Создание (POST)
- Получение всех (GET)
- Получение по ID (GET)
- Обновление (PUT)
- Удаление (DELETE)

### Бизнес-операции
- Подписаться / отписаться от пользователя
- Поставить / убрать лайк
- Просмотр подписчиков и подписок
- Лента постов (посты подписок)
- Количество лайков у поста
- Топ-5 постов по лайкам
